# 2、强调

## 2.1、加粗 `**`或者`__`包围

```
**兰州大学**
__兰州大学__
快捷键：Ctrl + B
```

**数学与统计学院**

## 2.2、斜体 `*`或者`_`包围

```
*兰大大学*
_兰州大学_
快捷键：Ctrl + I
```

*数学与统计学院*

## 2.3、斜体加粗 `***`或者`___`包围

```
***兰州大学***
___兰州大学___
快捷键：Ctrl + I + B
```

***数学与统计学院***

## 2.4、删除线 `~~`包围

```
~~兰州大学~~
应该可以选中文字同时在两边加上~~
```

~~数学与统计学院~~

## 2.5、高亮 `==`包围 （扩展语法）

```
==兰州大学==
```

==数学与统计学院==

数学

# 8、上下标（扩展语法）

```
^ 上标  包围内容
~ 下标  包围
```

比如

2^3^       2的3次方

H~2~O    水的化学式
